📚 Student Task Manager

A simple Student Task Manager web application built using React and Node.js.
It helps students organize and manage their daily tasks efficiently.

🚀 Features
  📝 Add new tasks
  📋 View all tasks
  ❌ Delete tasks
  📱 Clean and responsive UI

🛠️ Tech Stack
  Frontend: React.js ⚛️
  Backend: Node.js / Express 🚀
  Database: MongoDB (if used) 🍃
  API: Axios
  
📂 Projet Structure
  StudentTask-manager
  │
  ├── frontend
  ├── backend
  ├── models
  ├── routes
  └── README.md

⚙️ Installation
  1. Clone the repo
    git clone https://github.com/Shaik-Afreen10/StudentTask-manager
  2. Install frontend
    cd client
    npm install
    npm start
  3. Install backend
    cd server
    npm install
    node index.js
🎯 How it works
  - User adds a task
  - Task is stored in backend
  - Tasks are fetched and displayed in UI
  - User can delete tasks when completed
